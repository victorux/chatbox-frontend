## Setup

If you want to play around with ChatBox App, feel free to clone our repo. To start, please enter the following commands on your terminal:

```
git clone https://github.com/2009-fsa-cs-ashes-archers/JobMapApp
cd JobMapApp
npm install
npm run start-dev
```
